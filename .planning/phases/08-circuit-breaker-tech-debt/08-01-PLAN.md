<objective>
Wire circuit breaker and retry config into the preview command, add defensive `_init_breaker()` calls in `_init_database_searchers()`, and add BreakerConfig unit tests plus a preview wiring regression test.

Purpose: Preview searches currently bypass circuit breaker protection because `preview.py:37` constructs `Biocurator` without `global_breaker`/`global_retry` kwargs. The searcher constructors already call `_init_breaker()` (NCBI at `ncbi/searcher.py:40`, UniProt at `uniprot/searcher.py:37`), but since `global_breaker` is None when not passed, `_init_breaker()` returns None — no breaker is active. This plan closes that gap and adds regression tests to prevent future drift.

Output: Modified `preview.py` line 37, modified `curator.py` lines 70 and 81, BreakerConfig tests in `test_schema.py`, preview wiring regression test in new `test_preview.py`.
</objective>
