# Plan 04-02: `biocurator files` List Mode

## Objective

Implement `biocurator files [JOB_NAME] [--config PATH]` — display downloaded files for a specific job from its manifest, or show a summary of all jobs that have downloaded data. Render Rich tables. No checksum verification in this plan (that's Plan 04-03).

## Background

Each job's output directory (`job.export.outdir`) contains `manifest.json` written by `StreamingExporter`. Manifest schema: `{ manifest_version, job_name, generated_at, config, databases, stats.{total_records, total_files}, files: [{path, format, sha256, size, record_count, provider}] }`. The `size` field is in bytes. This plan builds the list display only; `--verify` is additive in 04-03.

## must_haves

```yaml
truths:
  - "biocurator files job-name --config {cfg}" exits 0 and displays a table containing each file from the manifest (path, format, size, records)
  - "biocurator files --config {cfg}" (no job name) exits 0 and shows a row for every job, indicating which have manifests
  - "biocurator files nonexistent --config {cfg}" exits non-zero with "Unknown job" in output
  - When a job has no manifest.json, "biocurator files job-name --config {cfg}" exits 0 with a message saying to run the job first
  - files_command is registered as app.command("files") in src/biocurator/cli/main.py
  - pytest tests/cli/test_files.py -q exits 0 (list-mode tests pass)
```

- Returns `f"{size_bytes / 1_048_576:.1f} MB"` if >= 1_048_576
- Returns `f"{size_bytes / 1024:.1f} KB"` if >= 1024
- Returns `f"{size_bytes} B"` otherwise

Function signature:
```
def files_command(
    job_name: Annotated[Optional[str], typer.Argument(help="Job name to inspect (omit for summary of all jobs)")] = None,
    config: Annotated[str, typer.Option("--config", "-c", help="Path to the YAML config file")] = "biocurator_config.yaml",
    verify: Annotated[bool, typer.Option("--verify", help="Re-read files and verify SHA-256 checksums")] = False,
) -> None:
```

Docstring: "List downloaded files for a job, or summarize all jobs with data. Use --verify to re-check checksums."

Error handling (same as status.py):
- `ConfigNotFoundError`: red error + `raise typer.Exit(1)`
- `InvalidConfigError`: red invalid config + `raise typer.Exit(1)`

**Mode 1 — `job_name` provided, `verify=False`:**
1. Find matching job in `global_config.jobs` by `job.name == job_name`. If not found: `console.print(f"[bold red]Unknown job:[/bold red] {job_name}. Available: {', '.join(j.name for j in global_config.jobs)}")` then `raise typer.Exit(1)`
2. `manifest_path = Path(job.export.outdir) / "manifest.json"`
3. If manifest doesn't exist: `console.print(f"[yellow]No data for job '{job_name}' — run it with `biocurator run` first.[/yellow]")` then `return` (exit 0)
4. Load manifest: `manifest = json.loads(manifest_path.read_text())`
5. Handle JSON error: `console.print(f"[bold red]Cannot read manifest:[/bold red] {manifest_path}")` then `raise typer.Exit(1)`
6. Build file table: `title=f"Files — {job_name}"`, `show_header=True`, `header_style="bold magenta"`
   Columns: `"Filename"` (style="cyan"), `"Format"`, `"Size"` (justify="right"), `"Records"` (justify="right"), `"SHA-256"`
   For each entry in `manifest["files"]`: row with `entry["path"]`, `entry["format"]`, `_format_size(entry.get("size", 0))`, `str(entry.get("record_count", "—"))`, `entry["sha256"][:12] + "..."`
7. Print table, then footer: `console.print(f"[dim]{manifest['stats']['total_files']} file(s), {manifest['stats']['total_records']} record(s) — generated {manifest['generated_at']}[/dim]")`

**Mode 2 — `job_name` is None, `verify=False`:**
1. Build summary table: `title="Downloaded Data"`, columns: `"Job"` (style="cyan"), `"Output Dir"`, `"Files"` (justify="right"), `"Records"` (justify="right"), `"Manifest"`
2. For each job in `global_config.jobs`:
   - `manifest_path = Path(job.export.outdir) / "manifest.json"`
   - If exists: load manifest, row with `job.name`, `job.export.outdir`, `str(manifest["stats"]["total_files"])`, `str(manifest["stats"]["total_records"])`, `"[bold green]✓[/bold green]"`
   - If not exists: row with `job.name`, `job.export.outdir`, `"[dim]—[/dim]"`, `"[dim]—[/dim]"`, `"[dim]none[/dim]"`
3. Print table
4. Count jobs with data. If zero: `console.print("[yellow]No downloaded data found. Run jobs with `biocurator run` first.[/yellow]")`

**`verify=True` branch:** In this plan, just placeholder — `if verify: console.print("[yellow]--verify not yet implemented in this plan[/yellow]"); return`. Plan 04-03 replaces this with real implementation.

Note: verify logic will be added in 04-03, which modifies this same file.

            }
        ],
    }
    (outdir / "manifest.json").write_text(json.dumps(manifest))
